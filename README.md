# Break Word Traps
Break Word Traps jest projektem stworzonym podczas HackYeah 2024. Jest to aplikacja pozwalająca na przeprowadzenie analizy wideo z przemową, która ma na celu sprawdzenie jej poprawności i zasugerowanie możliwych zmian.
# Moduły
Aplikacja korzysta z modułów zajmujących się poszczególnymi elementami analizy:
- analiza dźwięku
- analiza wideo
- analiza transkrypcji
